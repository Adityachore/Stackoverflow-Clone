## 🚀 How to Use

### Start Everything
```bash
# Terminal 1 - Backend
cd server
npm run dev

# Terminal 2 - Frontend  
cd stack
npm run dev
```

### Visit
```
http://localhost:3001
```

### See Real SO Data
- Scroll down on home page
- Look for orange cards
- These are real StackOverflow questions!
- Click to view original (opens StackOverflow.com)
